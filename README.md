# LittleBites-Project
🍔 Food Ordering Website

A responsive food ordering website built using HTML, CSS, and JavaScript.
This project includes a landing page, menu page, cart system, and checkout functionality.

🚀 Features
✅ Responsive Landing Page
🍽️ Menu Page with food items
🛒 Add to Cart functionality
❌ Remove items from cart
💰 Dynamic total price calculation
🧾 Checkout page with order placement
💾 Cart data stored using LocalStorage
🛠️ Tech Stack
HTML
CSS
JavaScript
